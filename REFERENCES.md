CoQuí References
------------------
**Last Updated:** Feb. 22, 2026

All calculations in CoQuí rely on **tensor hypercontraction (THC)** compression for the many-body Hamiltonian. Please cite the following foundational papers when using CoQuí:

- C.-N. Yeh, M. Morales, Low-Scaling Algorithm for the Random Phase Approximation Using Tensor Hypercontraction with k-point Sampling, [J. Chem. Theory Comput. 19, 18, 6197–6207 (2023)](https://pubs.acs.org/doi/abs/10.1021/acs.jctc.3c00615).
- C.-N. Yeh, M. Morales, Low-Scaling Algorithms for GW and Constrained Random Phase Approximation Using Symmetry-Adapted Interpolative Separable Density Fitting, [J. Chem. Theory Comput. 20, 8, 3184–3198 (2024)](https://pubs.acs.org/doi/abs/10.1021/acs.jctc.4c00085).

Depending on which feature(s) you use in CoQuí, please cite the relevant references below. 

**RPA correlation energy**
- C.-N. Yeh, M. Morales, Low-Scaling Algorithm for the Random Phase Approximation Using Tensor Hypercontraction with k-point Sampling, [J. Chem. Theory Comput. 19, 18, 6197–6207 (2023)](https://pubs.acs.org/doi/abs/10.1021/acs.jctc.3c00615).

**GW and constrained RPA (cRPA)**
- C.-N. Yeh, M. Morales, Low-Scaling Algorithms for GW and Constrained Random Phase Approximation Using Symmetry-Adapted Interpolative Separable Density Fitting, [J. Chem. Theory Comput. 20, 8, 3184–3198 (2024)](https://pubs.acs.org/doi/abs/10.1021/acs.jctc.4c00085).

**Second-order exchange (SOX) diagram with bare interactions (GF2 and GWSOX)**
- P. Pokhilko, C.-N. Yeh, M. Morales, D. Zgid, Tensor hypercontraction for fully self-consistent imaginary-time GF2 and GWSOX methods: theory, implementation, and role of the Green's function second-order exchange for intermolecular interactions, [J. Chem. Phys., 161, 084108 (2024)](https://doi.org/10.1063/5.0215954).

**Second-order screened exchange (SOSEX, 2SOSEX) and vertex-corrected GW self-energy (G3W2)**
- P. Pokhilko, C.-N. Yeh, M. Morales, D. Zgid, Tensor hypercontraction for self-consistent vertex corrected GW with static and dynamic screening; applications to molecules and solids with superexchange, [J. Chem. Phys., 162, 244110 (2025)](https://doi.org/10.1063/5.0269572).

**DIIS for SCF convergence**
- P. Pokhilko, C.-N. Yeh, D. Zgid, Iterative subspace algorithms for finite-temperature solution of Dyson equation, [J. Chem. Phys. 156, 094101 (2022)](https://doi.org/10.1063/5.0082586).

**GW+EDMFT**
- C.-N. Yeh, F. Petocchi, A. Hampel, P. Werner, O. Parcollet, A. Georges, M. Morales, Electronic correlations and dynamical screening with ab initio quantum embedding, [arXiv:2603.12336 (2026)](https://arxiv.org/abs/2603.12336).
